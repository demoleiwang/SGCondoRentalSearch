## Context

- 身份：NUS 博士后，刚从国内高校毕业过来，签了 2 年合同。
- 税前月薪：~SGD 6,500。扣完 CPF 实际 ~$5,200/月。
- 住了 3 个月 Airbnb，现在要正式租房。
- 目标：1BR 或 studio，预算 $2,500–$3,000/月（不想超过 50% 收入）。
- 工作地：NUS Kent Ridge 校区。走路或巴士 20 分钟内。
- 中国人，英文能交流但不流畅；网上搜索/看说明更习惯中文界面。
- 没有车，暂时也不买。

## Search Style

- 几乎只打中文，会夹一些英文地名/缩写（如 "NUS 附近"，"Clementi 1房"）。
- 查询偏口语化 / 不规范："有没有 NUS 附近 3000 以内 1 房"、"便宜一点 clementi 1bed"。
- 对数字敏感，会反复确认预算上限是否真的生效。
- 中文比例最高时可能直接打 "金文泰 1房 3000 以内朝南"，工具是否认得地名的中文写法很关键。
- 倾向先刷 3-5 个结果，再对比两个具体项目看细节。

## Success Criteria

1. 中文查询能被正确解析，价格和地区都不能漏。
2. 预算 ≤ $3,000 时结果不会"超预算塞进来"凑数。
3. 中文地名（金文泰 / 武吉知马 / 花拉公园）也认得，不只是 pinyin/拉丁化。
4. 结果里至少看得出哪些项目靠近公交站（不只是 MRT）。

## Quit Triggers

- UI 或关键提示只有英文，读着吃力。
- 明明说了 3000 以内，结果里依然出现 $3,500 的项目。
- 结果全是 condo，看不到同区域的 HDB 选项（他可能更适合 HDB 整租一间）。
- 中文地名打进去认不出来（如 "金文泰" 找不到 Clementi）。
